LeKiwi

A Low-Cost Mobile Manipulation Robot Built From Scratch

LeKiwi is a mobile manipulator that I built using custom 3D-printed components and roughly $1,000 of off-the-shelf hardware. The robot is compatible with Hugging Face’s LeRobot ecosystem and was designed to support real-world learning experiments involving perception, grasping, and object manipulation.

Robotics Transformers Imitation Learning OpenCV Python

Project Overview

LeKiwi is a low-cost mobile manipulation robot that I built from scratch as part of my UCLA Computer Engineering Capstone project, under the guidance of Professor Jonathan Kao. The robot integrates perception, grasping, and control using modern imitation-learning techniques within the Hugging Face LeRobot ecosystem.

I experimented with various tasks, but for my final project, I chose to have LeKiwi focus on collecting an empty water bottle. I teleoperated LeKiwi to collect real-world demonstrations and trained an Action Chunking with Transformers (ACT) policy to perform an end-to-end manipulation task: identifying an empty water bottle, grasping it without disturbing a nearby full bottle, and depositing it into a cart. Because of cheap, low-quality hardware, LeKiwi was not always consistent, and this demonstrates the tradeoff between reliability and operating on low-cost hardware.

Technical Stack

Key Features:

  • Custom-built hardware: 3D-printed frame, Raspberry Pi 5 control stack
  • Real-world demonstration collection via teleoperation
  • Vision-based object identification using ResNet-18
  • End-to-end manipulation using ACT imitation-learning policies
